# TUROS

Tiny Unfinished Roughly Operating System.


# Boot Loader

TUROS bootloader runs on bare-metal!

It doesn't requires any SDKs, like edk2.


# Kernel

TUROS kernel supports:

- KBC(Key Board Controller) Interrupt
- HPET(High Precision Event Timer)
